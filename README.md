# ROBOT_ARM
My first robotics project: A 4-axis robotic arm utilizing an electromagnet for object manipulation.

The components I used:
3 MGR996R servo motors (180°)
1 SG90 servo motor (180°)
1 5V Electromagnet (I found a module with it, without it you should use a transistor or a relay)
Wires
1 PCA9685 servo driver module
1 Arduino uno r3

To use it I created a python code, you can use what you want!
If you want to follow my tutorial you have to upload the .ino sketch to your arduino (which is wired by i2c to the servo driver) and use the .py file to use the robot arm.
